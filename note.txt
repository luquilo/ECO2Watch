catatan perintah git

git fetch = memperbarui repo local

git branch = mengecek kita ada di branch yang mana

git branch -a = melihat branch

git checkout {nama branch} = untuk pindah branch


// jika mau push codingan
- git add . = menambahkan codingan untuk siap di commit
- git commit -m {selesai mengerjakan beranda}
- git push origin {nama branch kita} = push code ke github
